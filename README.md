# mortonHx  
morton code just for 2D experimental haxe

```haxe  
var p = (cast( number ,Morton2D)).decode();
trace( p.x + ' ' p.y );
var p = new Point2DInt( 100, 100 );
var m: Mordon2D = p.toMordon2D();
```
